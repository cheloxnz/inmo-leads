<analysis>**original_problem_statement:**
The user wants to build a sophisticated WhatsApp bot for real estate agencies. The initial request was for a bot to qualify leads through a series of questions (intent, property type, zone, budget, urgency), score them, and notify human agents about hot leads.

Over the session, the scope expanded significantly based on user requests, turning the project into a multi-agent SaaS platform.

**PRODUCT REQUIREMENTS (PRD):**
- **Core Bot Flow:**
    - Greet user and ask for intent (buy, rent, invest).
    - Ask for the user's full name.
    - Ask for preferred zone, property type, number of bedrooms, and budget.
    - Validate budget: minimum of 0,000 for buy intent.
    - Ask about the urgency of the operation.
    - Schedule an appointment (date/time).
    - Score leads based on answers and classify them as hot, warm, or cold.
- **Agent/CRM Features:**
    - Web dashboard (React) to view and manage leads.
    - Email notifications to agents for hot leads via Gmail SMTP.
    - Ability to **Edit** (name, appointment date) and **Delete** leads from the dashboard.
    - A section for agents to add **internal notes** to each lead.
    - **Filter** the leads list (by name, zone, date) and **Export** to CSV/Excel.
- **Multi-Agent System (Requested, In Progress):**
    - **Authentication:** A login system for different users (Admins, Agents).
    - **Roles:** Admins can see everything; Agents can only see their assigned leads.
    - **Agent Management:** Admins can create and manage agent profiles (specialty, zones).
    - **Automatic Lead Assignment:** A hybrid system that assigns hot leads to agents based on specialty (buy/rent), zones, and current workload.
    - **Agent-Specific Dashboards:** Show metrics like conversion rates per agent.
    - **Real-time Push Notifications:** Notify agents on the dashboard about new leads, customer replies, and upcoming appointments.
- **Integrations:**
    - WhatsApp Cloud API.
    - Google Calendar for appointments.
    - Google Sheets for lead logging.
    - OpenAI for conversation intelligence.
